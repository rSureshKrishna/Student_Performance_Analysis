# Student_Performance_Analysis
🎓 Student Performance Analysis using Python. This project analyzes 300 students’ marks using NumPy, Pandas, and Matplotlib with visualizations like charts and heatmaps. Includes ML models (Linear Regression, Decision Tree) to predict performance and grades.
# 🎓 Student Performance Analysis using Python

## 📌 Overview

This project focuses on analyzing student academic performance using **Python-based data analysis and visualization techniques**. The dataset consists of **300 students' marks** across multiple subjects, and the analysis aims to extract meaningful insights, visualize patterns, and apply basic machine learning models.

---

## 📂 Dataset Details

The dataset includes the following attributes:

* Student_ID
* Student_Name
* Maths
* Physics
* Chemistry
* English
* Computer_Science
* Total (calculated)

Additional derived columns:

* Average Marks
* Grade (A, B, C, D, F)
* Result (Pass/Fail)

---

## 🛠️ Technologies Used

* Python 🐍
* NumPy (Numerical Analysis)
* Pandas (Data Manipulation)
* Matplotlib (Data Visualization)
* Scikit-learn (Machine Learning)

---

## 📊 Features & Analysis

### 🔍 Data Analysis

* Statistical summary (mean, median, standard deviation)
* Subject-wise performance comparison
* Top-performing students identification

### 📈 Data Visualization

* Bar charts (subject averages)
* Pie charts (grade distribution)
* Histograms (marks distribution)
* Scatter plots (subject relationships)
* Box plots (outlier detection)
* Heatmaps (correlation analysis)
* Stacked and line charts (performance trends)

### 🤖 Machine Learning

* **Linear Regression** to predict total marks
* **Decision Tree Classifier** for grade prediction
* Model evaluation using:

  * Mean Absolute Error (MAE)
  * R² Score
